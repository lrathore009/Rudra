# Web Research

When the owner asks you to research a topic:

1. Call the `web_search` tool with a focused query. Run 2–3 searches from different angles if the topic is broad.
2. Cross-check key facts against at least two sources; note where sources disagree.
3. Synthesize a briefing with these sections:
   - **Executive Summary** (2–3 sentences)
   - **Key Findings** (bulleted, each tied to a source)
   - **Open Questions** (what remains uncertain)
   - **Sources** (numbered, with URLs)
4. Mark anything uncertain explicitly. Never invent sources, dates, or figures.
